Общие параметры, используемые в проекте

Группа содержит описания параметров, которые используются в нескольких классах и функциях проекта.

Данные параметры не являются реальными глобальными переменными, а служат для централизованного документирования смысловых характеристик аргументов функций и методов.

amountOfWheels

Должно быть неотрицательным целым числом. Для некоторых видов транспорта может быть равно 0.

amountOfPassengers

Определяет максимальное количество пассажиров, которое может перевозить транспортное средство.

speed

Единицы измерения: километры в час (км/ч). Значение должно быть неотрицательным.

isSport

Используется в классах Motorcycle и Car.

Значения:

  • true — транспорт спортивный
  • false — транспорт обычный

Нахождение скорости:

\[v_{avg} = \frac{S}{T} \]

где:

  • \(v_{avg}\) — средняя скорость транспорта
  • \(S\) — пройденное расстояние
  • \(T\) — время движения
